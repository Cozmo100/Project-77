a. Why Do We Use Console?

➼ We use console.log to "test" or "run something" before we put it on html

➼ It is suggested to write in console before you print on html (suggested)




b. How do I set up Visual Studio Code?

➼ 1: Download and install Visual Studio to your corrosponding device

➼ 2: *read* the terms of policy, and accept

➼ 3: go to Folders and create an empty folder

➼ 4: Click "File" at the top of your taskbar, and click new folder

➼ 5: add "index.html" file and "style.css" file

➼ 6: Now you can start coding!




c. How do I Print Stuff on Console?

➼ 1: make an index.html file in Visual Studio

➼ 2: write down the basic html code

➼ 3: Click "Go Live" in the bottom right corner of your screen

➼ 4: Right click on your mouse onto the screen. Then select "inspect"

➼ 5: At the top of the screen, click console.

➼ 6: Start from there!
